# Basiscursus Copilot - Stad & Ambassadeurs

Dit project bevat de bronbestanden voor de interne Copilot-opleiding van de stad, gebouwd met MkDocs en het Material for MkDocs thema.

## Structuur
Alle inhoudelijke cursusbestanden bevinden zich in de `/docs` map.
- `/docs/medewerkers`: Basismateriaal en use-cases per stadsdienst.
- `/docs/ambassadeurs`: Train-the-trainer gidsen en beleidsrichtlijnen.

## Website
De cursus wordt automatisch gebouwd en gehost via GitHub Pages bij elke nieuwe push naar de `main` branch.
